Where does my Data go?
We’ve also designed the software with an intuitive and familiar representation of the necessary data – generic term we use for each is Data Entity – while some specific types are: Material, Process, Batch Plan, and Impurity. You can find a complete list of the types of Data Entities here.
How do I use SimpliDATA?
We’ve designed SimpliDATA to be … yes, you guessed it! Simple! The application interface generally follows a layout that you can find Application Layout here
What is SimpliDATA for?
Beyond it’s application to technology transfer, SimpliDATA serves as a process development tool, allowing developers to accumulate data over years and serving as an aid to establishing the Control Strategy and identifying the Critical Process Parameters. *SimpliDATA is designed and supported by experienced process developers and pharmaceutical manufacturers for use by biotech innovators and contract manufacturers.
